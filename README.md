# Institute of Digital Risk – Homepage

## Project Overview

This project is a responsive single-page website designed for the **Institute of Digital Risk (IDR)**. The website introduces IDR’s mission, services, and community while providing a clear call to action for users to explore programs or register their interest.

The goal of the website is to present IDR as an **industry-led training and deployment institute focused on digital, cyber, and technology risk roles**.

---

## Features

The webpage includes the following sections:

### 1. Hero Section

* Displays the main headline: **“Advancing the Future of Digital Risk.”**
* Includes a short description of IDR’s mission.
* Contains a call-to-action button encouraging users to explore programs.
* Uses a background image with an overlay for better readability.

### 2. About IDR

* Explains IDR as an **industry-led training institute**.
* Highlights collaboration between **academic institutions and industry projects**.

### 3. Service Pillars / Model

The website presents three core service areas:

* **Academy** – Training programs and bootcamps for students and professionals.
* **Innovation & Incubation** – Development of intellectual property and AI governance frameworks.
* **Advisory Services** – Helping organisations implement digital risk frameworks such as **NIST, ISO 27001, and NIS2**.

### 4. Community

Describes the community served by IDR:

* Students
* Early-career professionals
* Practitioners in sectors such as **financial services and critical infrastructure**.

### 5. Contact / Register Interest

A simple contact form allows users to submit:

* Name
* Email
* Message

---

## Technical Implementation

### Technologies Used

* **HTML5**
* **CSS3**
* **Vanilla JavaScript**

### Key Technical Features

* Semantic HTML structure (`header`, `nav`, `main`, `section`, `footer`)
* Responsive layout using **Flexbox**
* **Sticky navigation bar**
* **Smooth scrolling navigation**
* Hover effects for buttons and links
* Accessible colour contrast
* Google Fonts for typography
* Custom **SVG cube logo**

No external CSS frameworks such as Bootstrap were used.

---

## Logo Design

The logo uses a **cube-inspired geometric icon** representing:

* Structure
* Risk management
* Resilience

The colour palette uses:

* **Orange**
* **Black**
* **White**

Two logo variants were created:

* **Icon only**
* **Icon with text: “Institute of Digital Risk”**

The SVG format ensures the logo remains **sharp and scalable across devices**
